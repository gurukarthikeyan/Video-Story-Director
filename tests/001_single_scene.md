# Test 001 - Single Scene

## Purpose

Verify that the Story Director can generate one complete, high-quality scene with a Frame 0 prompt and a matching LTX prompt.

---

## User Prompt

I want a single 10-second scene of a cute baby dragon discovering a glowing butterfly in a magical forest. Make it funny and Pixar-inspired.

---

## Expected Behavior

- Generates a Frame 0 prompt.
- Generates exactly one LTX scene prompt.
- Character appearance is fully established.
- Environment is clearly established.
- Story is simple and complete.
- Motion is natural.
- Camera is cinematic.
- Emotional progression is clear.
- Prompt is optimized for LTX Video.

---

## Evaluation Checklist

- [ ] Correct number of scenes
- [ ] Frame 0 generated
- [ ] Character consistency
- [ ] Environment consistency
- [ ] Motion quality
- [ ] Camera quality
- [ ] Emotional clarity
- [ ] LTX suitability
- [ ] Prompt clarity
